# Typography Link

#### Overview

Typography.Link provides an inline hyperlink that inherits typography styles. It is ideal for contextual navigation or secondary actions that live inside copy.

## Link Styles

Links come in two styles:

### Default Link (Underlined)

By default, links are **underlined with black text**. This style is used for primary navigation within content.

<Canvas of={LinkStories.Primary} />

### Link Without Underline

Links can be displayed **without underline in blue** by setting `underline={false}`. This style uses the theme's info color and includes hover states.
<Canvas of={LinkStories.WithoutUnderline} />
