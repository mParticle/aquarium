# FloatButton

### Overview

The **FloatButton** is a fixed-position action trigger that floats above page content, typically in the bottom-right corner. It is commonly used for primary quick actions like help, feedback, or "back to top".

FloatButton uses `position: fixed` — it is **not** placed inside layout containers like Flex or Grid. It renders as a viewport overlay.
