# RadioCard

### Overview

RadioCards present a set of mutually exclusive options as selectable cards, each with a title, optional description, and a radio indicator. Use them when choices need more context than a standard radio group can provide.

### Vertical Layout

Stack cards vertically when each option has a longer description or when screen width is limited.

<Canvas of={RadioCardStories.Vertical} />

### Horizontal Layout

Arrange cards side-by-side for compact option sets with shorter labels.

<Canvas of={RadioCardStories.Horizontal} />

### Disabled Options

Individual options can be disabled with an optional tooltip explaining why.
