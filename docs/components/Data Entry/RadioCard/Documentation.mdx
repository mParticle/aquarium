# RadioCard

#### Overview

The **RadioCard** component wraps radio selection in a card layout, giving each option more visual weight and space for descriptive content. It supports vertical and horizontal orientations and handles disabled states at both the individual option and group level.

### When to Use

Use **RadioCard** when:
- Users choose exactly one option from a set and each option benefits from a title and supporting description.
- The choice is consequential enough to warrant a more prominent layout than a standard radio button.
- You want to visually differentiate options side by side (horizontal) or stacked (vertical).
