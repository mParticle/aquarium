# Flex

A flexible container built on CSS flexbox. Use `Flex` instead of raw `<div style={{ display: 'flex' }}>` so direction, gap, alignment, and wrapping all flow through the same API — and spacing always lands on a design token.

## When to use

- Arranging elements in a row or column with consistent gap.
- Centering, end-aligning, or distributing content along either axis.
- Wrapping rows of items without writing custom media queries.

## Props at a glance

- `vertical` — switches to a column layout.
- `gap` — accepts any size token (`SizeSm`, `SizeLg`, …) or number.
- `justify` / `align` — all standard flex values.
- `wrap` — lets children flow to the next line when space runs out.
