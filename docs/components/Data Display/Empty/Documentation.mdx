# Empty

#### Overview

The **Empty State** component provides feedback to users when there is no data or content to display. It often includes a message, icon, and possibly an action button to guide users on what to do next.

### When to Use
Use an Empty State when:
- A user lands on a page with no data yet, such as an empty dashboard or list view.
- A search returns no results.
- A filter or selection has no matching data.
