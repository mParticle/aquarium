# Tag

#### Overview

The Tags component displays small, labeled elements used to categorize, identify, or highlight content. Tags are versatile for indicating types or other identifiers and can include text, icons, or color coding to enhance readability and organization.


### Info Tags
Used to indicate contextual information, such as environments or states. These tags are static, with the exception of optional additional info displayed on hover.

#### Current Usage

- [**Tables Usage:**](https://mparticle.github.io/aquarium/?path=/story/components-data-display-tag--table-usage) Used to indicate environment (e.g., production or development), priority, or health status.  

### Clicable Tags

Used as interactive elements that can function as filters or selection tools.


#### Current Usage
- **Search Modal**: Allows users to filter search results by clicking on a tag.
