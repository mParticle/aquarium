# Badge

#### Overview

Small numerical value or status indecator for UI elements.


#### When to use 

Use **Badge** when a status needs to be displayed next to an element in a compact form. For display status with accompanying text (e.g., "Active," "Failed," or "Draft"), use **[Tags](https://mparticle.github.io/aquarium/?path=/story/components-data-display-tag--primary)**.
