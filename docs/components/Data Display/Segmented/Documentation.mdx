# Segmented

#### Overview

The **Segmented** component is a compact switch that lets users pick one mode at a time.

#### When to use

Use **Segmented** for switching between a small set of related modes, such as:

- Cards vs. list view
- Criteria vs. code mode  
- Edit vs. code mode

#### When to avoid

- There are many modes
- A toggle or checkbox is enough
- For displaying multiple options (e.g., days of the week)
