# HelpIcon

### Overview
The HelpIcon is a contextual help indicator that displays a tooltip with additional information when hovered. It combines the RoktHelpCircle icon with a Tooltip into a single, easy-to-use component.

### When to use:
- To provide inline contextual help next to labels, form fields, or settings.
- To explain technical terms or metrics without cluttering the interface.
- To offer tips or guidance that users can discover on hover.

### Differences from IconWithTooltip
- **HelpIcon** is a simplified, opinionated component specifically for help/contextual info. It always uses the RoktHelpCircle icon.
- **IconWithTooltip** is more flexible and allows any icon to be paired with a tooltip.

Use `HelpIcon` when you need a standard help indicator. Use `IconWithTooltip` when you need a different icon.
