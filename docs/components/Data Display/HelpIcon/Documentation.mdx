# Help Icon

### Overview
A contextual help indicator that displays a tooltip when hovered. Use `IconWithTooltip` with the `RoktHelpCircle` icon to create this pattern.

### When to use:
- To provide inline contextual help next to labels, form fields, or settings.
- To explain technical terms or metrics without cluttering the interface.
- To offer tips or guidance that users can discover on hover.

### Usage

```tsx
import { IconWithTooltip } from '@mparticle/aquarium'
import { RoktHelpCircle } from '@mparticle/aquarium/icons'

<IconWithTooltip
  title="Helpful context about this field"
  iconProps={{ name: RoktHelpCircle }}
/>
```
