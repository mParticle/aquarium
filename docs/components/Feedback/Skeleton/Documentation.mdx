# Skeleton

A **Skeleton** is a placeholder loading state that mimics the structure of content before actual data loads, improving perceived performance and reducing layout shifts.

#### When to use
- The content has a predictable structure (e.g., tables, lists, cards).
- You want to give users a visual cue of what to expect while waiting for data.
