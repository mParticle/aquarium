# Alert

#### Overview

The **Alert** component is used to display important messages, warnings, or confirmations to users. It helps provide contextual feedback in response to user actions or system events.

### When to Use
- When you need to show alert messages to users.
- When you need a persistent, static container that can be dismissed by user actions.
- Use Message if the content should dismiss automatically
