# Spin

A **Spin** is a simple loading indicator, often displayed as a spinning animation, used to signal an ongoing process such as data fetching or submission.

#### When to use
- Loading a whole page or dashboard, where content structure is not as relevant.
- Indicating a **short, ongoing process** like API requests or form submissions.
