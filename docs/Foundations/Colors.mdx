These represent the core identity of the brand and are used to focus the user on core actions, such as primary buttons, navigation elements, and key call-to-actions.

#### Secondary
These colors complement the primary palette and are used to support the primary actions without drawing too much attention, typically applied to backgrounds, borders, and secondary UI components.











Border Colors: Colors used for UI borders, such as disabled states.
Icon Colors: Colors applied to icons, including strong and brand variations.
